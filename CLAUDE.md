# Prime IPTV – Project Instructions

## Branch Strategy

**Website changes** (HTML, CSS, JS files in the root website directories:
`/`, `/en/`, `/fr/`, `/es/`, `/pt/`, `/de/`, `/tr/`, `/it/`, `/nl/`,
`/images/`, `/guide/`, `sitemap.xml`, etc.) must always be committed and
pushed **directly to `main`** — no feature branches.

**Flutter app changes** (anything inside `prime-player/`) follow the normal
feature-branch workflow assigned per session.
